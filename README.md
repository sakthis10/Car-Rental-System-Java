Car Rental System (Java + MySQL)
Console-based car rental management system built using Java, JDBC and MySQL.

Features
Add car
View cars
Rent car
Return car
Data stored in MySQL database
Technologies
Java
OOP
JDBC
MySQL
GitHub
How to Run
Clone the repo
Compile: javac -cp ".;mysql-connector-j-9.6.0.jar" *.java
Run: java -cp ".;mysql-connector-j-9.6.0.jar" Main
